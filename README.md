# Proyecto-API-bancarias-y-Tests
Proyecto API bancaria, desarrollado con Node.js, Express y PostgreSQL. Permite gestionar clientes, cuentas y transacciones, implementando lógica de negocio como depósitos, retiros y validación de saldo. Incluye pruebas con Postman y endpoints de análisis para verificar la consistencia de datos y simular escenarios reales de sistemas financieros.
